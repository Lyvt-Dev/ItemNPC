### 🌀 LyvtNPC - Modern & Interactive Item NPCs

**LyvtNPC** is a lightweight, modern Minecraft server plugin that allows you to create interactive, rotating floating items as NPCs! Say goodbye to laggy, old-school armor stands and hello to sleek, highly optimized `ItemDisplay` entities. Perfect for server lobbies, crate areas, spawn points, or minigame selectors.

✨ **Key Features:**

* **Interactive & Command-Ready:** Assign any command to an NPC. When a player clicks the rotating item, the command is executed instantly.
* **Modern & Lag-Free:** Uses the latest `ItemDisplay` and `Interaction` entities instead of heavy armor stands, keeping your server performance top-notch.
* **Eye-Catching Animation:** NPCs feature a smooth, continuous rotation to easily grab your players' attention.
* **All-in-One Command:** Manage everything seamlessly in-game via `/npc <spawn|delete|reload>` with full smart tab-completion.
* **Highly Customizable:** Easily adjust the size, location, and material of your NPCs on the fly.
* **Built-in Spam Protection:** Features an automatic 3-second click cooldown to prevent players from spamming commands.
* **Plug & Play:** Fully internationalized (English) with a clean, easy-to-read configuration file (`npcs.yml`).

**Commands & Permissions:**

* `/npc spawn <Name> <Size> <Item> <Command>` - *(lyvtnpc.spawn)*
* `/npc delete <Name>` - *(lyvtnpc.delete)*
* `/npc reload` - *(lyvtnpc.reload)*
